# 🏆 王者荣耀英雄数据分析

基于 Python 的王者荣耀英雄数据分析项目。

## 📖 项目简介

本项目以王者荣耀英雄数据为研究对象，利用 Python 对英雄属性数据进行清洗、统计分析和可视化展示。

通过对不同定位英雄的初始生命、初始法力、初始物攻、初始物防等属性进行分析，探索英雄之间的差异特征，并使用多种图表直观展示分析结果。

本项目旨在实践数据分析的完整流程。

---

## 🎯 分析目标与研究问题

本项目以王者荣耀英雄属性数据为研究对象，通过 Python 对数据进行清洗、统计分析和可视化展示，探索不同定位英雄之间的属性差异及其设计特点。

### ❓ 研究问题

#### 1. 不同定位英雄的属性是否存在明显差异？

王者荣耀中的英雄被划分为坦克、战士、刺客、法师、射手和辅助等不同定位。不同定位承担着不同的游戏职责，因此其基础属性也应具有一定差异。

本项目通过分组统计和柱状图分析，对各定位英雄的初始生命值、初始法力值、初始物理攻击和初始物理防御等属性进行比较，观察不同定位之间的数值分布特征。

#### 2. 哪类英雄整体生存能力最强？

英雄的生存能力通常与生命值和防御属性密切相关。

通过对各定位英雄的生命值和防御属性进行统计分析，探究不同定位在生存能力方面的差异，从而了解游戏角色设计中的定位特点。

#### 3. 是否存在明显偏离同定位平均水平的英雄？

部分英雄为了突出个性化设计，可能在某些属性上显著高于或低于同定位英雄。

本项目利用箱线图和异常值分析方法，识别各定位中的特殊英雄，探索游戏数值设计中的差异化特征。

#### 4. 英雄属性之间是否存在相关关系？

英雄的生命值、法力值、攻击力和防御力等属性并非完全独立。

通过计算属性相关系数并绘制散点图，分析不同属性之间的关联程度，探索英雄数值体系的整体规律。

## 🔍 分析方法

本项目采用探索性数据分析（Exploratory Data Analysis，EDA）方法开展研究，主要包括以下步骤：

1. 数据清洗与预处理

   * 检查缺失值与重复值
   * 异常数据处理

2. 描述性统计分析

   * 均值分析
   * 最大值与最小值分析
   * 中位数分析
   * 标准差分析

3. 分组统计分析

   * 按英雄定位进行分类统计
   * 比较每种定位之间的属性差异

4. 数据可视化分析

   * 饼状图
   * 柱状图
   * 箱线图
   * 散点图

通过上述方法，对王者荣耀英雄属性数据进行全面分析，并总结不同定位英雄的数值特征。

## 🔎 分析结论

根据对王者荣耀英雄属性数据的统计分析与可视化结果，可以得到以下结论：

### 1. 不同定位英雄在属性分布上存在明显差异

不同定位英雄的生命值、法力值、攻击力和防御力表现出显著不同的分布特征，说明游戏在角色设计过程中充分考虑了各定位的职责需求。

### 2. 坦克英雄整体生存能力最强

从生命值和防御属性来看，坦克英雄普遍具有更高的生命值和防御能力，体现了其承担前排承伤任务的定位特点。

### 3. 法师英雄拥有较高的法力属性

法师英雄整体法力值水平明显高于其他定位，符合依赖技能输出的角色特征。

### 4. 射手英雄呈现高输出、低生存的特点

射手英雄通常具有较高的输出能力，但生命值和防御能力相对较低，体现出高风险、高收益的设计思路。

### 5. 存在少量属性异常英雄

通过箱线图分析发现，部分英雄的属性明显偏离同定位平均水平。这些英雄通常具有特殊技能机制或独特玩法，体现了游戏角色设计中的差异化策略。

### 6. 英雄属性之间存在一定正相关关系

通过散点图和相关系数分析发现，英雄的生命值与防御属性之间呈现一定程度的正相关关系。

> **相关系数约为：** `r = 0.4030`

这表明生命值较高的英雄通常也具备较好的防御能力，但这种相关关系并不十分显著，说明英雄属性设计还受到定位、技能机制等多种因素的影响。

趋势线结果进一步表明，两项属性总体呈上升趋势，但仍存在较大的个体差异。
