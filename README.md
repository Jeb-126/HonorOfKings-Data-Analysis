# 🏆 王者荣耀英雄数据分析

基于 Python 的王者荣耀英雄数据分析项目。

## 📖 项目简介

本项目以王者荣耀英雄数据为研究对象，利用 Python 对英雄属性数据进行清洗、统计分析和可视化展示。

通过对不同定位英雄的初始生命、初始法力、初始物攻、初始物防等属性进行分析，探索英雄之间的差异特征，并使用多种图表直观展示分析结果。

本项目旨在实践数据分析的完整流程，包括：

* 数据获取与整理
* 数据清洗
* 数据统计分析
* 数据可视化
* 项目结构化管理
